# PIINs

This repository is dedicated to the **study and exploration of Physics-Informed Neural Network (PINN) architectures**, with a focus on understanding their structure, training strategies, and application to physics-based problems governed by differential equations.

## Authorship and supervision
This study was conducted by **Danilo Di Fábio Bueno**, under the orientation of **Prof. Dr. Vanderlei Cunha Parro**.

## Institutional

This work is developed within the **NSSE (Núcleo de Sistemas Eletrônicos Embarcados)**, a research group from **Instituto Mauá de Tecnologia (IMT)**.

## Repository content
This repository may include:
- Source code for PINN implementations  
- Notebooks for experimentation and visualization  
- Numerical and analytical comparisons  
- Documentation and supporting materials
